# AI工具雷达

一个面向中文用户的原创 AI 工具导航网站原型，参考成熟 AI 工具导航站的信息架构与浏览体验，但使用自有品牌、原创文案、精选数据和可替换的公共 favicon 图标来源。

## 运行

直接打开 `index.html` 即可运行。也可以在项目目录启动本地静态服务：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 当前能力

- 顶部导航、桌面固定左侧分类栏、移动端分类抽屉
- Hero 大搜索、顶部搜索、移动搜索同步筛选
- 热门推荐、连续工具分类分区
- 16 个主分类、约 80 个精选工具、分类内子筛选
- 工具访问外链、收藏、最多 3 个工具对比、详情预览抽屉
- 提交工具、购买推荐位、会员订阅、行业报告、社群、企业线索等商业入口
- 统一合作表单，本地模拟提交反馈
- 响应式布局、键盘焦点、弹窗关闭和 reduced-motion 支持

## 商业闭环建议

- 把提交工具和 Featured 推荐位接入表单服务或 CRM，形成厂商线索池。
- 将收藏、对比、价格监控和深度评测包装成会员权益。
- 用分类浏览、搜索词和合作表单生成企业采购意向线索。
- 增加工具详情页和 SEO 内容页，承接搜索流量与投放落地页。
- 后续可迁移到 CMS 或数据库，把 `app.js` 中的数据拆成可运营内容。
